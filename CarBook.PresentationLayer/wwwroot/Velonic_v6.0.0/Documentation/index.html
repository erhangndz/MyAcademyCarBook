Introduction

Velonic is a fully featured premium admin template built on top of awesome Bootstrap 5.3.0. It's crafted using modern web technologies HTML5, CSS3 and jQuery. The theme includes beautiful dashboard, many ready to use hand crafted components, UI elements, widgets, pages, etc. The components could be used very easily on any page. The theme is fully responsive and customizable.

We really care for our buyers and so in case if you are having any question or feedback, please feel free to get back to us via support.

Structure

Extract the zip file you received after purchase and you would find the exact below files and folders:

└── README.md
